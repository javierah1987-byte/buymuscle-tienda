# buymuscle-tienda

Tienda de suplementación deportiva - BuyMuscle Sevilla

## Stack
- Next.js 14 + TypeScript
- Supabase (PostgreSQL + Auth)
- Vercel (Deploy)

## Páginas
- / — Homepage con categorías y novedades
- /tienda — Catálogo con filtros
- /producto/[id] — Ficha de producto
- /carrito — Carrito de compra
- /distribuidores — Portal distribuidores
